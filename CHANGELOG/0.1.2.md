## AirModel 0.1.2: Field metadata moves to AirField

AirModel's `Field` function lives in the [AirField](https://github.com/feldroy/AirField) package as `AirField`. Import it directly instead of through AirModel.

```sh
uv add AirModel>=0.1.2
```

### What's changed

- **`from airmodel import Field` becomes `from airmodel import AirField`**, and `Field(...)` calls become `AirField(...)`. `AirField` is re-exported from both `airmodel` and `airfield`.
