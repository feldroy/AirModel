# AirModel

Async ORM for Pydantic models and PostgreSQL, with a Django-inspired query API. Pair it with [Air](https://airwebframework.org/) for zero-config database connections, Pydantic-based form validation, and HTMX interactions, all in a single-file app with no boilerplate.
